# usage

## run in dev mode

```
$ make run
```

## run in release mode

```
$ make release-run
```

## build

```
$ make run
```

## test

```
$ make test
```

## format

```
$ make fmt

## build container

```
$ make build-container
```

## run container

```
$ make run-container
